## Installation

1. Copy this plugin directory to your ZephyrGate plugins directory
2. Add the plugin path to your `config.yaml`:

```yaml
plugins:
  paths:
    - "path/to/plugins"
```

3. Enable the plugin in your `config.yaml`:

```yaml
plugins:
  enabled_plugins:
    - demo_plugin
```

4. Configure the plugin (optional):

```yaml
plugins:
  demo_plugin:
    enabled: true
    
    task_interval: 300  # 5 minutes
    
    
```

## Configuration

See `config_schema.json` for the complete configuration schema.

### Configuration Options

- `enabled` (boolean): Enable or disable the plugin (default: true)

- `task_interval` (integer): Interval for periodic task in seconds (default: 300)
